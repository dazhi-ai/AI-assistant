## 1. 服务器部署步骤（源码方式，非 Docker）

### 1.1 拉取小智后端源码
```bash
cd /opt
git clone https://github.com/xinnan-tech/xiaozhi-esp32-server.git
```

### 1.2 安装运行环境（按官方推荐 conda）
```bash
conda create -n xiaozhi-esp32-server python=3.10 -y
conda activate xiaozhi-esp32-server
cd /opt/xiaozhi-esp32-server/main/xiaozhi-server
pip install -r requirements.txt
```

### 1.3 配置 Doubao 模型
1. 在 `data/` 目录创建 `.config.yaml`
2. 参考 `scripts/xiaozhi/.config.yaml.doubao.example` 填写至少以下字段：
   - `selected_module.LLM: DoubaoLLM`
   - `LLM.DoubaoLLM.base_url: https://ark.cn-beijing.volces.com/api/v3`
   - `LLM.DoubaoLLM.model_name: doubao-1-5-pro-32k-250115`
   - `LLM.DoubaoLLM.api_key: 你的ARK_API_KEY`
3. 或直接使用脚本模板快速生成（Linux）：
```bash
bash scripts/xiaozhi/deploy-xiaozhi-linux.sh /opt/xiaozhi-esp32-server 你的ARK_API_KEY
```

### 1.4 启动验证
```bash
conda activate xiaozhi-esp32-server
cd /opt/xiaozhi-esp32-server/main/xiaozhi-server
python app.py
```

日志出现以下关键信息可判定启动成功：
- OTA 地址：`http://<server-ip>:8003/xiaozhi/ota/`
- WebSocket 地址：`ws://<server-ip>:8000/xiaozhi/v1/`

## 2. systemd 托管

### 2.1 服务文件
- 模板文件：`scripts/systemd/xiaozhi-server.service`

### 2.2 安装命令
```bash
sudo cp /opt/ai-assistant/scripts/systemd/xiaozhi-server.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable xiaozhi-server
sudo systemctl start xiaozhi-server
sudo systemctl status xiaozhi-server
```

## 3. 与本项目联动建议
