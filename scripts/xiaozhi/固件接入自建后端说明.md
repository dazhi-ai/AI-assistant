# 小智固件接入自建后端说明（任务 3.1 前置）

## 1. 已完成的固件源码修改
固件源码中 OTA 默认地址已改为自建后端占位值：

- 文件：`services/xiaozhi-esp32-firmware/main/Kconfig.projbuild`
- 配置项：`CONFIG_OTA_URL`
- 默认值：`http://YOUR_SERVER_IP:8003/xiaozhi/ota/`

说明：设备开机会先请求 OTA 地址，再由 OTA 下发 websocket 配置（`/xiaozhi/v1/`）。

## 2. 你需要手动替换的值
- `YOUR_SERVER_IP`：替换成你的服务器局域网 IP，例如 `192.168.1.25`

最终示例：
- `http://192.168.1.25:8003/xiaozhi/ota/`

## 3. 编译前建议
如果你使用 ESP-IDF：
1. 进入固件目录：`services/xiaozhi-esp32-firmware`
2. 执行：`idf.py menuconfig`
3. 在 `Xiaozhi Assistant -> Default OTA URL` 再次确认地址是否正确

## 4. 烧录后联调检查
设备连接网络后，串口日志中应出现：
- OTA 接口地址（`http://<IP>:8003/xiaozhi/ota/`）
- WebSocket 地址（`ws://<IP>:8000/xiaozhi/v1/`）

若地址仍为官方地址，说明编译时未使用当前源码配置，请清理后重新编译。
